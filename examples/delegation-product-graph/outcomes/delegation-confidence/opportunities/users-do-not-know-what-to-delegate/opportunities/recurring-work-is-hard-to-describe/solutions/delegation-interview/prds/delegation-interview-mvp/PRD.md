# Delegation Interview MVP

```yaml
type: prd
id: prd.delegation-interview-mvp
status: example
```

Build the smallest flow that interviews a user about one recurring task and turns the answers into a reusable agent instruction artifact.

## Acceptance Criteria

- The interview anchors on one concrete recent task.
- The output includes inputs, steps, edge cases, quality bar, and stop-and-ask conditions.
- The output can be used by an agent in a fresh context.
