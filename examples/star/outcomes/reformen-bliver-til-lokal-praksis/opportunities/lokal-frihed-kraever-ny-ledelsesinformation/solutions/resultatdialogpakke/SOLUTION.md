# Resultatdialogpakke

```yaml
type: solution
id: solution.resultatdialogpakke
status: planned
```

En pakke med nøgletal, dialogspørgsmål og eksempler på lokal opfølgning, der hjælper kommuner med at bruge STARs data i ledelsesdialog frem for kun kontrol.

## Produktrisici

- Value: ledere kan opleve materialet som endnu et styringslag.
- Usability: indikatorer skal være få, forklarlige og koblet til handling.
- Feasibility: kræver robuste datadefinitioner og aftaler om sammenlignelighed.
- Viability: kan gøre statslig opfølgning mere nyttig for lokal forbedring.

## Antagelser

- Kommunale ledere ønsker nationale sammenligninger, hvis de kan bruges til lokal læring.
- Dialogspørgsmål kan reducere risikoen for snæver KPI-styring.
