# Resultatdialogpakke

En pakke med nøgletal, dialogspørgsmål og eksempler på lokal opfølgning, der hjælper kommuner med at bruge STARs data i ledelsesdialog frem for kun kontrol.

## Hvorfor Det Kan Virke

Den forbinder nationale data med lokale ledelsesbeslutninger og gør opfølgning mere anvendelig som læring, prioritering og tidlig støtte.

## Produktrisici

- Værdi: ledere kan opleve materialet som endnu et styringslag.
- Brugbarhed: indikatorer skal være få, forklarlige og koblet til handling.
- Gennemførlighed: kræver robuste datadefinitioner og aftaler om sammenlignelighed.
- Bæredygtighed: kan gøre statslig opfølgning mere nyttig for lokal forbedring.

## Antagelser

- Kommunale ledere ønsker nationale sammenligninger, hvis de kan bruges til lokal læring.
- Dialogspørgsmål kan reducere risikoen for snæver KPI-styring.
